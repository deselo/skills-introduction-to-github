# 五子棋游戏

这是一个使用 C++ 实现的简单五子棋游戏，可以在 Windows 系统上运行。

## 编译和运行

### 方法一：使用 MinGW (g++)
1. 安装 MinGW 并确保 g++ 在系统路径中
2. 打开命令提示符，进入此目录
3. 运行命令：`g++ main.cpp -o gomoku.exe`
4. 运行游戏：`gomoku.exe`

### 方法二：使用 Visual Studio
1. 打开 Visual Studio
2. 创建一个新的空项目
3. 将 main.cpp 添加到项目中
4. 编译并运行项目

## 游戏规则
1. 游戏在 15x15 的棋盘上进行
2. 黑棋 (X) 先手，白棋 (O) 后手
3. 率先在横、竖或对角线上连成 5 子的一方获胜
4. 如果棋盘下满仍未分出胜负，则为平局

## 操作方法
1. 运行程序后，会显示棋盘
2. 按照提示输入行号和列号（例如：3 5）
3. 程序会自动检查胜负并切换玩家
4. 游戏结束后，按任意键退出
