# 代码导航扩展 for VS Code

增强大型项目的代码导航功能 - 快速跳转到函数调用和定义。

## 功能特性

- **函数定义导航**：快速跳转到任何函数的定义
- **使用追踪**：查找函数在项目中的所有使用位置
- **调用层次可视化**：在webview面板中查看函数的调用层次
- **项目扫描**：自动扫描项目中的函数定义和调用
- **数据持久化**：保存导航数据以加快访问速度
- **智能缓存**：缓存导航数据以提高性能
- **多语言支持**：支持C/C++、Python、Java、JavaScript和TypeScript
- **代码结构视图**：显示当前文件的函数、类和变量结构
- **符号搜索**：搜索项目中的所有符号
- **依赖分析**：显示文件的包含和导入依赖
- **文件比较**：比较两个文件的差异
- **书签管理**：管理代码中的书签
- **代码度量**：显示文件的行数、函数数、类数等统计信息

## 安装

### 从VSIX文件安装

1. 下载最新的 `.vsix` 文件
2. 在VS Code中，打开 `扩展` (Ctrl+Shift+X)
3. 点击右上角的 `...` 菜单
4. 选择 `从VSIX安装...`
5. 选择下载的 `.vsix` 文件

### 手动安装

1. 克隆或下载此仓库
2. 在VS Code中打开文件夹
3. 运行 `npm install` 安装依赖
4. 按 `F5` 在新的VS Code窗口中启动扩展

## 使用方法

### 命令

扩展提供以下命令：

- **查找函数定义** (`codeNavigation.findDefinition`)：查找所选函数的定义
- **查找函数使用** (`codeNavigation.findUsages`)：查找所选函数的所有使用位置
- **显示调用层次** (`codeNavigation.showCallHierarchy`)：显示所选函数的调用层次
- **扫描项目导航** (`codeNavigation.scanProject`)：扫描项目中的函数定义和调用
- **清除导航缓存** (`codeNavigation.clearCache`)：清除缓存的导航数据
- **显示代码结构** (`codeNavigation.showCodeStructure`)：显示当前文件的代码结构
- **搜索符号** (`codeNavigation.searchSymbols`)：搜索项目中的符号
- **显示依赖分析** (`codeNavigation.showDependencies`)：显示文件的依赖关系
- **比较文件** (`codeNavigation.compareFiles`)：比较两个文件的差异
- **添加书签** (`codeNavigation.addBookmark`)：在当前位置添加书签
- **显示书签** (`codeNavigation.showBookmarks`)：显示所有书签
- **显示代码度量** (`codeNavigation.showCodeMetrics`)：显示文件的代码度量信息

### 键盘快捷键

- `F12`：查找函数定义
- `Shift+F12`：查找函数使用
- `Ctrl+Shift+H`：显示调用层次
- `Ctrl+Shift+O`：显示代码结构
- `Ctrl+T`：搜索符号
- `Ctrl+Shift+D`：显示依赖分析
- `Ctrl+F2`：添加书签
- `Shift+F2`：显示书签
- `Ctrl+点击`：跳转到函数定义

### 上下文菜单

在编辑器中右键点击所选函数名，可访问导航命令：
- 查找函数定义
- 查找函数使用
- 显示调用层次
- 显示代码结构
- 搜索符号
- 显示依赖分析
- 添加书签

## 工作原理

1. **项目扫描**：扩展扫描项目中的函数定义、调用、符号和依赖关系
2. **数据存储**：导航数据存储在JSON文件中，以便快速访问
3. **实时更新**：扩展每小时自动重新扫描项目
4. **智能导航**：当您选择函数名时，扩展会找到其定义或使用位置

## 支持的语言

- Python

## 性能

- **初始扫描**：大型项目可能需要几分钟
- **后续扫描**：由于缓存，速度更快
- **内存使用**：将导航数据存储在内存中以便快速访问
- **磁盘使用**：导航数据存储在全局存储目录的JSON文件中

## 配置

扩展在首次激活时自动扫描项目并缓存结果。您可以使用 `扫描项目导航` 命令手动触发重新扫描。

### 扫描间隔

扩展每小时自动重新扫描项目（未来版本可配置）。

### 缓存位置

导航数据存储在：
- Windows: `%USERPROFILE%\.vscode\extensions\code-navigation-extension\navigationData.json`
- macOS/Linux: `~/.vscode/extensions/code-navigation-extension/navigationData.json`

## 故障排除

### 常见问题

1. **未找到定义**：函数可能未在项目中定义，或者项目可能需要重新扫描
2. **性能缓慢**：对于非常大的项目，考虑清除缓存并重新扫描
3. **结果不准确**：扩展使用正则表达式模式，可能无法捕获所有边缘情况

### 日志

扩展将其活动记录到VS Code输出面板。要查看日志：
1. 转到 `查看` > `输出`
2. 从下拉菜单中选择 `扩展主机`

### 清除缓存

如果遇到缓存数据问题：
1. 运行 `清除导航缓存` 命令
2. 使用 `扫描项目导航` 重新扫描项目
